# MasterGo 动效插件

一款专为MasterGo设计的动效插件，帮助UI设计师在设计稿中快速添加丰富的交互效果。

## 🌟 主要功能

### ✨ 效果类型
- **点击效果**：缩放、颜色变化、波纹、弹跳等多种点击交互
- **轮播效果**：滑动轮播、淡入淡出、3D翻转等展示效果
- **悬停效果**：元素悬停时的淡入、高亮等效果
- **过渡效果**：页面元素的滑入、出现等动画效果

### 🎯 核心特性
- 🔄 **实时预览**：所见即所得的效果预览
- 📋 **代码导出**：支持CSS、JavaScript、完整HTML三种格式导出
- ⚙️ **参数配置**：灵活的效果参数调节
- 🎨 **直观界面**：简洁易用的操作界面

## 🚀 快速开始

### 安装依赖
```bash
npm install
```

### 开发模式
```bash
npm run dev
```

### 生产构建
```bash
npm run build
```

### 启动服务器
```bash

```

## 📖 使用指南

### 1. 选择效果
在主界面的"常用效果"区域点击想要的效果卡片

### 2. 调整参数
在"参数设置"面板中调整动画时长、缓动函数等参数

### 3. 实时预览
在"预览"标签页中查看效果的实际表现

### 4. 导出代码
在"导出"标签页中选择格式并导出代码

## 🛠 技术架构

### 核心模块
- **PluginCore**：插件核心管理器，负责状态管理和事件系统
- **UIManager**：用户界面管理器，负责界面渲染和交互
- **EffectManager**：效果管理器，负责效果的注册和应用
- **PreviewSystem**：预览系统，提供实时效果预览
- **CodeExporter**：代码导出器，生成可复用的代码

### 效果组件
- **ClickEffect**：点击效果组件
- **CarouselEffect**：轮播效果组件

### 工具函数
- **Utils**：通用工具函数集合

## 🎨 支持的效果

### 点击效果
- `click-scale`：点击缩放效果
- `click-color-change`：点击颜色变化
- `click-ripple`：点击波纹效果
- `click-bounce`：点击弹跳效果

### 轮播效果
- `carousel-slide`：滑动轮播
- `carousel-fade`：淡入淡出轮播
- `carousel-flip`：3D翻转轮播

### 其他效果
- `hover-fade`：悬停淡入效果
- `slide-in`：滑入过渡效果

## 📤 导出格式

### CSS格式
生成纯CSS动画代码，适用于静态效果实现

### JavaScript格式
生成交互式JavaScript代码，适用于动态效果实现

### HTML格式
生成完整的可运行HTML页面，包含样式和脚本

## 🔧 配置选项

### 全局配置
- `animationDuration`：默认动画时长（毫秒）
- `easingFunction`：默认缓动函数
- `autoPreview`：是否启用实时预览
- `codeExportFormat`：默认导出格式

### 效果参数
每种效果都有其特定的可配置参数，如：
- 缩放比例
- 颜色值
- 动画时长
- 过渡效果等
